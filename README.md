# Global Maternal Mortality EDA

This project presents an exploratory data analysis (EDA) of global maternal mortality data using Python. It explores trends, regional differences, and key indicators through data cleaning, visualization, and statistical analysis.

## Tools
- Python
- Pandas
- NumPy
- Matplotlib
- GeoPandas

## Contents
- Data cleaning and preprocessing
- Exploratory data analysis
- Data visualizations
- Geographic analysis
